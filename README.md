# rust_os
Testing x86-64 osdev with Rust and nasm.

## Current features:
* Text output terminal
* Long mode with Rust

## Planned in near future:
* Virtual TTYs
* Keyboard input
* Memory manager and paging

## Not-in-so-near future features:
* Filesystem
* Disk IO
* Networking
* Executable programs

# Running
Have qemu and vagrant installed, and run `./autoboild.sh -u`.
